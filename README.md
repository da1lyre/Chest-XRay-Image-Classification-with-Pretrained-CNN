# Chest-XRay-Image-Classification-with-Pretrained-CNN
Классификация рентгеновских снимков грудной клетки для обнаружения пневмонии с использованием Transfer Learning.

## Результаты

| Метрика | Значение |
|---------|----------|
| Точность на валидации | **92.47%** |
| Точность на тесте | **83.65%** |

### Детальный отчет по классам

| Класс | Precision | Recall | F1-Score | Поддержка |
|-------|-----------|--------|----------|-----------|
| NORMAL | 0.96 | 0.59 | 0.73 | 234 |
| PNEUMONIA | 0.80 | **0.99** | **0.88** | 390 |

**Macro avg:** Precision 0.88, Recall 0.79, F1 0.81  
**Weighted avg:** Precision 0.86, Recall 0.84, F1 0.83

### Интерпретация результатов

- **Recall на классе PNEUMONIA = 99%** — модель находит **почти всех больных** пневмонией, что критически важно для медицинской диагностики
- **Precision на классе NORMAL = 96%** — если модель говорит "здоров", то это почти всегда верно
- **Ложные срабатывания** (41% здоровых ошибочно определены как больные) являются приемлемым компромиссом, так как направляют пациентов на дополнительное обследование

## Архитектура модели

- **Backbone:** ResNet34, предобученная на ImageNet
- **Вход:** Ч/б рентгеновский снимок
- **Выход:** 2 класса (NORMAL / PNEUMONIA)
- **Регуляризация:** Dropout (0.4)
